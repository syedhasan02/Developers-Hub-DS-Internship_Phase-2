# Term Deposit Subscription Prediction (Bank Marketing)
Overview

This project predicts whether a bank customer will subscribe to a term deposit based on marketing campaign data. The goal is to analyze customer behavior and build a machine learning model that can help banks improve their marketing strategies.

Dataset
The dataset used in this project is the Bank Marketing Dataset from the UCI Machine Learning Repository.
It contains customer information such as:

- Age
- Job
- Marital status
- Balance
- Contact type
- Previous campaign outcomes
- And more

Technologies Used
  Python
  Pandas
  NumPy
  Matplotlib
  Seaborn
  Scikit-learn

Project Workflow
 1. Loaded and explored the dataset 
 2 Cleaned and preprocessed the data
 3. Encoded categorical features
 4. Split data into training and testing sets
 5. Trained Logistic Regression model
 6. Evaluated the model using:
    - Accuracy
    - Confusion Matrix
    - Classification Report
    - ROC Curve

Results
The model achieved around 90% accuracy in predicting customer subscriptions. The project also helped in understanding which customer patterns are more likely to lead to successful term deposit subscriptions.

Conclusion
This project demonstrates how machine learning can support banks in making smarter marketing decisions by identifying potential customers more effectively.
