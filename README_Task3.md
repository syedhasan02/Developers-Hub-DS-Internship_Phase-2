# Energy Consumption Time Series Forecasting

## Overview
This project focuses on forecasting short-term household energy consumption using historical electricity usage data. The aim is to analyze time-based patterns and predict future energy usage using different time series forecasting models.
The project compares ARIMA, Prophet, and XGBoost models to identify the most effective forecasting approach.

## Dataset
Dataset Used: Household Power Consumption Dataset

The dataset contains:
- Date and Time
- Global Active Power
- Voltage
- Global Intensity
- Sub-metering values

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels
- Prophet
- XGBoost

## Project Workflow
1. Loaded and cleaned the dataset
2. Parsed date and time columns
3. Resampled time series data
4. Performed Exploratory Data Analysis (EDA)
5. Engineered time-based features
6. Applied ARIMA forecasting
7. Applied Prophet forecasting
8. Applied XGBoost forecasting
9. Compared model performance using MAE and RMSE
10. Visualized actual vs forecasted energy consumption

## Results
The forecasting models successfully predicted future household energy usage patterns. Model comparison helped identify which algorithm performed better for short-term forecasting.

This project demonstrates how machine learning and time series analysis can be used to forecast household energy consumption and support better energy management decisions.
